Options 選項顯示
簡介
感覺下拉列表的方式很Low？小按鈕觸屏點擊有夠麻煩？
現在全屏顯示自定義的選項，適用於觸屏，炫酷的動畫來了！
使用方法
格式: 
$.Options({
	<選項一>:<函數一>,
	<選項二>:<函數二>
	[,....]
},<標題/問題>);
  • 至少有一個選項
  • 標題可選，樣式自適應
示例
代碼: 
	//無標題選項
	$.Options({
		"「OK，沒問題，包在我身上」":function(){
			$("#fb").text("你「OK，沒問題，包在我身上」");
		},
		"「呃……恐怕不行誒」":function(){
			$("#fb").text("你「呃……恐怕不行誒」");
		}
	})

	//有標題選項
	$.Options({
		"天依藍":function(){
			$("#fb").text("你喜歡的顏色是「天依藍」");
		},
		"橙色":function(){
			$("#fb").text("你喜歡的顏色是「橙色」");
		},
		"白色":function(){
			$("#fb").text("你喜歡的顏色是「白色」");
		},
		"取消":function(){
			$("#fb").text("你取消了選項");
		}
	},"哪一個是你喜歡的顏色")
