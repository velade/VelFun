CustomContextmenu 客製滑鼠右鍵選單
簡介
客製滑鼠右鍵選單是爲部分需要自定義右鍵選單而設計的功能，
使用方法
格式: 
$(<元素>).CustomContextmenu({"[icon(<圖標路徑>)]Title":<函數>[,…]});
  • icon(<圖標路徑>)為顯示在選單前的圖標，路徑無須加引號，可以省略
  • Title爲選單選項，必須具有
  • 函數是點擊後執行的動作
示例
代碼: 
	$("#test0").CustomContextmenu({
		'A選項':function () {
			$.Msgbox('選擇了A選項','結果');
		},
		'B選項':function () {
			$.Msgbox('選擇了B選項','結果');
		},
		'C選項':function () {
			$.Msgbox('選擇了C選項','結果');
		}
	})

右鍵彈出選單
