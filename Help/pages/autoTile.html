autoTile 自動平鋪
簡介
很多時候我們需要一個平鋪的布局，京東和淘寶的商品展示就是一個典型的應用。但是往往會遇到排列間距這個頭痛的問題，儘管div是可以自動換行的，但正因如此，常常出現每行最後一個不足一個元素，而導致的右邊空白一塊。對於這個問題，我們提供了解決方案：autoTile！
使用方法
格式: 
	$(<元素>).autoTile(<是否修正間距>);
  • 是否修正間距：這是一個用於指示元素是否有多餘的間距的參數(如inline-block)，布爾型變量，為true時通過float布局清除多餘的間距。
  • 為什麼稱inline-block的間距是多餘的？因為inline-block的間距盡管官方說這是符合標準的，但這是一個糟糕的設計，因為它不可控，在不同瀏覽器有不同的間距。也不可獲取，這意味它除了破壞布局以外，沒有任何意義，無論規則是什麼，它都該改變，可控或可獲取。否則這只是個多餘的東西，因為我們無法使用它。
示例
代碼: 
	html:
	<div class="box">
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
		<div class="item"></div>
	</div>

	CSS:
	.box{
		display: block;
		width: 500px;
		height: 500px;
		overflow: auto;
		border: 1px solid black;
	}
	.box::-webkit-scrollbar{
		display: none;
	}
	.box > .item{
		display: inline-block;
		width: 100px;
		height: 100px;
		border: 1px solid black;
		margin: 10px auto;
	}

	jQuery:
	$('.item').autoTile(true);
