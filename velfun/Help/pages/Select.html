UL模擬SELECT
簡介
自帶的select控件在我們看來一直是一個被忽略的BUG，它存在太多的缺陷，例如readonly無效（有人說可以disbaled但是這樣提交表單是不會提交它的），樣式不能變更，下拉箭頭樣式不統一，選項間距太小等。因此很多人選擇模擬方式，但每一個都重寫費時費力。
現在我們解決了這個問題，我們使用更加方便的方式幫助你用ul模擬select
使用方法
在要ul添加客制屬性select即可
例如: 
<ul name="名字" style="樣式" select>
	<li value="值" selected>顯示名</li>
	<li value="值" >顯示名</li>
	<li value="值" >顯示名</li>
</ul>
我們盡可能貼近select的選項定義方式，但並未包含默認樣式，因此ul的樣式定義是必須的
此功能屬於智慧型方法，無需輸入JS代碼

相關方法

示例
代碼: 
<ul name="s1" select style="width:200px; height:30px;">
	<li value="0" selected>選項0</li>
	<li value="1" >選項1</li>
	<li value="2" >選項2</li>
</ul>
